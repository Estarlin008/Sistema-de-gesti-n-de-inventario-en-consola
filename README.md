
Sistema de gestión de inventario

Para completar este reto, tendrá que crear una aplicación de consola en la que los usuarios puedan gestionar el stock 
de productos. Los usuarios deberán poder añadir nuevos productos, actualizar el stock y eliminar productos.

Algunas características clave incluyen:
Añadir nuevos productos con nombre, precio y cantidad de stock.
Actualizar las existencias cuando los productos se venden o se reponen.
Ver todos los productos y sus niveles de existencias.
Eliminar productos del inventario.

Requisitos funcionales:
	
  1. El sistema debe permitir ingresar un producto nuevo con nombre, precio y cantidad de stock.
	
  2. El sistema debe permitir ingresar más stock de un producto ya existente.
	
  3. El sistema debe de permitir vender un producto existente.
	
  4. El sistema debe mostrar el stock del producto.
	
  5. El sistema debe permitir eliminar un producto del inventario.
	
  6. El sistema debe permitir ver los productos eliminados del inventario.
	
Requisitos no funcionales:
	
  1. El sistema debe funcional en consola.
	
  2. Debe validar entradas de información.
	
  3. Debe de mostrar el menú claramente.
	
  4. Debe evitar de que el programa ce cierre si el usuario no lo indica.

Objetivos:
	
  1. Desarrolla una aplicación de consola en C#.
	
  2. Aplicar estructura de control y bucles con coherencias.
	
  3. Implementar métodos reutilizables.
	
Estructura del proyecto:
	
  1. Tarea 1: Método que muestre el menú.
	
  2. Tarea 2: Leer opciones del menú.
	
  3. Tarea 3: Ejecutar opción según lo seleccionado.
	
  4. Tarea 4: Agregar producto.
	
  5. Tarea 5: Ver stock de producto.
	
  6. Tarea 6: Eliminar producto.
	
  7. Tarea 7: Agregar más stock de producto.
	
  8. Tarea 8: Vender producto. 
	
	
Variable principal:

Clase del producto (Nombre, Precio, Cantidad).

Switch.

For/ foreach.

If/else.

While.

arreglo dinámico vinculado a la clase.
